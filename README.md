Стек: Java, Java HTTP Server, Java HttpClient, Google Gson, JUnit, Maven.
Что сделано: Разработал сетевое приложение с раздельной архитектурой клиента и сервера. Настроил кастомную маршрутизацию входящих запросов через HTTP-хендлеры, реализовал корректную обработку REST-методов (GET, POST, DELETE) с отдачей валидных семантических кодов ответов (например, 201 Created). На стороне клиента реализовал десериализацию сложных структур данных из формата JSON с помощью библиотеки Gson.
